本包说明文件

array           : 基础数组 以及Arrays工具类的常用方法
basic           : 基础 循环语句,条件控制语句 if else foreache for ,类和对象的基础支持 变量,修饰符等
charchar        : 字符相关的内容 以及字符类常用的检测方法,是否是字符,大小写转换
dateAndTime     : 时间,日期,日历相关的操作
exception       : 异常的捕获和抛出  自定义异常类的操作
inAndOutStream  : 输入输出流,用二进制流的形式读取和修改文件内容
mathAndNumber   : 常用计算 数据类型的包装类 随机数的生成
method          : java中的方法 传参 返回值 静态方法 对象方法
scanner         : 控制台输入类的操作
stringBuffer    : 字符串的操作 (可变字符串)
stringstring    : 字符串类的操作 常用方法 (拼接,过滤空格,比较,获取长度)
